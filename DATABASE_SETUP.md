# 데이터베이스 설정

이 프로젝트는 SQLite 데이터베이스를 사용하여 플레이어 데이터, 힌트, 거래 로그를 영구 저장합니다.

## 설치

```bash
npm install better-sqlite3
```

## 데이터베이스 파일

- 데이터베이스 파일: `game_data.db` (프로젝트 루트에 자동 생성)
- 서버 시작 시 자동으로 테이블이 생성됩니다.

## 저장되는 데이터

1. **플레이어 데이터**
   - 닉네임, 현금, 총 자산
   - 연습 모드 / 실제 게임 모드 구분

2. **플레이어 주식**
   - 각 플레이어가 보유한 주식 수량

3. **플레이어 힌트**
   - 힌트 난이도, 내용, 가격, 라운드 정보

4. **거래 로그**
   - 주식 매수/매도
   - 포인트 추가
   - 힌트 구매

## 데이터베이스 구조

- `players`: 플레이어 기본 정보
- `player_stocks`: 플레이어 주식 보유량
- `player_hints`: 플레이어 힌트 목록
- `transactions`: 모든 거래 로그

## 주의사항

- 서버 재시작 시에도 데이터가 유지됩니다.
- 게임 초기화 시 데이터베이스는 초기화되지 않습니다 (필요시 수동 삭제).
- `game_data.db` 파일을 삭제하면 모든 데이터가 초기화됩니다.
